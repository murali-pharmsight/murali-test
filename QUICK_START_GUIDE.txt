                            HOW TO GET STARTED
═══════════════════════════════════════════════════════════════════════════════

STEP 1: Navigate to the project folder
─────────────────────────────────────────
Open terminal/command prompt and go to:
/path/to/meeting-notes-app

STEP 2: Install dependencies
─────────────────────────────
Run this command to install all required packages:
$ npm install

(This will take 2-5 minutes. Be patient!)

STEP 3: Start development server
─────────────────────────────────
Run this command to start the app:
$ npm start

Your browser will automatically open to http://localhost:3000

STEP 4: Start using the app!
────────────────────────────
- Click "Start Recording" to begin recording a meeting
- Enter meeting details (optional)
- Your audio will be automatically captured
- When done, click "Stop Recording"
- Summary generates automatically
- View, edit, and export your meetings

                          TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Problem: npm install fails
Solution: 
  1. Clear npm cache: npm cache clean --force
  2. Delete package-lock.json: rm package-lock.json
  3. Try again: npm install

Problem: Port 3000 already in use
Solution:
  1. Kill the process using port 3000
  2. Or specify a different port: PORT=3001 npm start

Problem: Microphone permission denied
Solution:
  1. Check browser permissions for the site
  2. Reset site permissions in browser settings
  3. Try a different browser if needed

Problem: App not loading
Solution:
  1. Clear browser cache (Ctrl+Shift+Delete)
  2. Close all tabs and reopen
  3. Check browser console (F12) for errors
  4. Try a different browser
